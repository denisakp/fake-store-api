# Delete an Order

This method allows a user to remove an order from the database based on its ID. If the request is successful,
the API responds with a status code of 204. In case of validation errors, such as an invalid ObjectId,
the API returns a 422 status code. Additionally, if the provided ID does not correspond to any order in the database,
the API returns a 400 status code.

```js copy showLineNumbers filename="orders/delete-order.js"
const axios = require('axios');

let config = {
  method: 'delete',
  url: 'http://localhost:3000/api/orders/6485d7a45562163f95c6e3e2',
  headers: {
    'Accept': 'application/json'
  }
};

axios.request(config)
.then((response) => {
  console.log(JSON.stringify(response.data));
})
.catch((error) => {
  console.log(error);
});

```

<Callout>
    Upon execution, you will receive No Content (HTTP 204 code). However,
    please note that no changes will be made to the actual database.
</Callout>
