# Load all categories

This method allows a user to retrieve the list of categories. If the operation is successful,
the API responds with an array of Category models and an HTTP status code of 200. In case of an error,
the API returns a status code of 500 and includes a "message" property describing the error message.


```js copy showLineNumbers filename="categories/fetch-all.js"
const axios = require('axios');

let config = {
  method: 'get',
  url: 'https://fakestoreapi.denisakp.dev/api/categories',
  headers: {
    'Accept': 'application/json'
  }
};

axios.request(config)
.then((response) => {
  console.log(JSON.stringify(response.data));
})
.catch((error) => {
  console.log(error);
});
```
