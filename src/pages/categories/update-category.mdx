# Update a category

This method allows a user to update a category based on its ID. If the operation is successful,
the API responds with a response containing a status code of 200 and the model with the new values.

In case of an error, such as missing required properties or an invalid ID (not a valid ObjectId),
the API returns a 400 status code. For validation errors in the request, the API returns a 422 status code.

```js copy showLineNumbers filename="categories/patch-category.js"
const axios = require('axios');
let data = JSON.stringify({
  "name": "updated"
});

let config = {
  method: 'patch',
  url: 'https://fakestoreapi.denisakp.dev/api/categories/6479065cb337e71eb79da693',
  headers: {
    'Accept': 'application/json',
    'Content-Type': 'application/json'
  },
  data
};

axios.request(config)
.then((response) => {
  console.log(JSON.stringify(response.data));
})
.catch((error) => {
  console.log(error);
});

```

<Callout>
    Upon execution, you will receive an object with the sent ID containing updated values. However,
    please note that no changes will be made to the actual database.
</Callout>

Here's an example of the API's response when everything goes smoothly.

```json copy filename="API Response"
{
    "id": "6479065cb337e71eb79da693",
    "name": "updated"
}
```
