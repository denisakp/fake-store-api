# Update password

This method in the API allows a user to change their password.
The user needs to provide the current password along with the new password.
The API validates that the provided current password matches the one stored in the database and that the new password meets the requirement of at least 8 alphanumeric characters.

If the password update is successful, the API responds with HTTP status code 204. In case the new password does not meet the requirements, a 400 error code is returned. For validation errors, the API returns a 422 error code.


```js copy showLineNumbers filename="auth/password.js"
const axios = require('axios');
let data = JSON.stringify({
  "current": "Fake:strore@password",
  "password": "1p@sswor{]"
});

let config = {
  method: 'patch',
  url: 'https://fakestoreapi.denisakp.dev/api/auth/password',
  headers: {
    'Accept': 'application/json',
    'Content-Type': 'application/json'
  },
  data
};

axios.request(config)
.then((response) => {
  console.log(JSON.stringify(response.data));
})
.catch((error) => {
  console.log(error);
});

```

<Callout>
    Upon execution, you will receive No Content (HTTP 204 code). However, please note that no changes will be made to the actual database.
</Callout>
