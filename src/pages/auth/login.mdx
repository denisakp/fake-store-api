# Login: Authenticate your users


The login method allows you to authenticate users using an email address and password.
To create a more realistic API, we have included form data validation and error handling.


For instance, if a user provides an invalid email address, they will receive a validation error message with an HTTP 400 status code,
along with a descriptive error message. Similarly, if they provide an incorrect email/password combination, they will also receive an HTTP 400 status code with an error message. However, if the combination is correct, a token will be generated and included in the response.

```js copy showLineNumbers filename="auth/login.js"
const axios = require('axios');
let data = JSON.stringify({
  "email": "john.doe@site.com",
  "password": "Fake:strore@password"
});

let config = {
  method: 'post',
  url: 'http://localhost:3000/api/auth/login',
  headers: {
    'Accept': 'application/json',
    'Content-Type': 'application/json'
  },
  data : data
};

axios.request(config)
.then((response) => {
  console.log(JSON.stringify(response.data));
})
.catch((error) => {
  console.log(error);
});

```
