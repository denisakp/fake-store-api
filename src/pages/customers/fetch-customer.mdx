# Fetch a single Customer

This method allows a manager to retrieve a customer based on his ID. The API responds with a response containing a status
code of 200 if the product object exists. If the provided ID is not valid or does not correspond to an existing customer,
the API returns a 400 status code. Additionally, if the manager submits an ID that is not a valid ObjectId,
the API returns a 422 status code.

```js copy showLineNumbers filename="customers/fetch-one.js"
const axios = require('axios');

let config = {
  method: 'get',
  url: 'http://localhost:3000/api/customers/6485c35814c402ee08ec6294',
  headers: {
    'Accept': 'application/json'
  }
};

axios.request(config)
.then((response) => {
  console.log(JSON.stringify(response.data));
})
.catch((error) => {
  console.log(error);
});

```
