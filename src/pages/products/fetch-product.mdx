# Fetch a single Product

This method allows a user to retrieve a product based on its ID. The API responds with a response containing a status
code of 200 if the product object exists. If the provided ID is not valid or does not correspond to an existing product,
the API returns a 400 status code. Additionally, if the user submits an ID that is not a valid ObjectId,
the API returns a 422 status code.

```js copy showLineNumbers filename="products/fetch-one.js"
const axios = require('axios');

let config = {
  method: 'get',
  url: 'http://localhost:3000/api/products/64858a3114c402ee08ec6293',
  headers: {
    'Accept': 'application/json'
  }
};

axios.request(config)
.then((response) => {
  console.log(JSON.stringify(response.data));
})
.catch((error) => {
  console.log(error);
});

```
