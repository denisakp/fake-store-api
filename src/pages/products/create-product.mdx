# Create a Product

The "createProduct" method allows a user to register a new product in the database. If the operation is successful,
the API responds with a response containing a status code of 201. In case of validation errors,
such as missing required properties of the product object, the API returns a 422 status code.

```js copy showLineNumbers filename="products/create-product.js"
const axios = require('axios');
let data = JSON.stringify({
  "name": "Ball",
  "price": 2800,
  "description": "Product description here",
  "stock": 8,
  "images": [
    "http://placeimg.com/640/480/food",
    "http://placeimg.com/640/480",
    "http://placeimg.com/640/480",
    "http://placeimg.com/640/480"
  ],
  "categories": [
    "iphone",
    "smartphone"
  ],
  "sku": "2023060001"
});

let config = {
  method: 'post',
  url: 'https://fakestoreapi.denisakp.dev/api/products/',
  headers: {
    'Accept': 'application/json',
    'Content-Type': 'application/json'
  },
  data
};

axios.request(config)
.then((response) => {
  console.log(JSON.stringify(response.data));
})
.catch((error) => {
  console.log(error);
});


```

> By sending an object similar to the code provided, you will receive an object with a new ID. However,
please be aware that no real data will be inserted into the database. Therefore, attempting to access the new ID will result in a 404 error.
